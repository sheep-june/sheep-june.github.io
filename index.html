Shopping Mall - 商品購入サイト（Node.js + React）

▶ 実際のサイトはこちら
バックエンド | フロントエンド
🔧 設計資料（Figma）: ※ 一部変更された部分があります
▶ プロトタイプ
▶ 開発用設計

Express をバックエンド、React (Vite + Tailwind CSS) をフロントエンドに採用した、 商品検索・カート・決済・レビュー機能を備えたフルスタック EC プラットフォームです。

🎯 主な機能

  • ユーザー認証（JWT）: 会員登録・ログイン・ログアウト
  • 商品管理: 登録・一覧・検索・絞り込み・詳細表示
  • カート操作: 追加・削除・数量変更・購入処理
  • 購入履歴: 履歴一覧・合計金額表示
  • レビュー投稿・表示: 購入済みユーザーのみ許可
  • ウィッシュリスト: お気に入り登録・解除・一覧表示
  • お問い合わせ・FAQ: 投稿・取得
  • 管理者認証: 管理者ログイン
  • 管理者ダッシュボード: ユーザー管理・投稿管理・広告バナー管理
  • 広告バナー管理（管理者向け CRUD）: バナー作成・更新・削除・一覧表示

🚀 デプロイ

  • フロントエンド: Vercel による自動デプロイ
  • バックエンド: AWS EC2 上で PM2 管理、Nginx + Certbot で HTTPS 運用
  • CI/CD パイプライン: GitHub Actions を使用し、プッシュ時にテスト → ビルド → 自動デプロイを実行

🔄 処理フロー

ユーザー → React UI → Axios API call → Express Server → JWT認証 → Mongooseモデル → MongoDB
